# 🎨 AI 썸네일 생성 워크플로우

## 사용 방법

1. **유튜브 URL 제공**
   - 유튜브 URL을 붙여넣기
   - 예: "이 영상 썸네일 만들어줘: https://youtube.com/watch?v=xxx"

2. **AI가 분석 및 생성**
   - 영상 제목/썸네일 분석
   - 주제에 맞는 새 이미지 생성
   - 창의적인 문구 작성
   - 3개 버전 제공

3. **확인 후 선택**
   - ✅ 마음에 들면: "1번 다운로드" 또는 "저장해줘"
   - 🔄 수정 원하면: "더 밝게", "문구 바꿔줘", "다른 스타일로"
   - ❌ 처음부터: "다시 만들어줘"

## 요청 예시

```
이 영상 썸네일 만들어줘: [URL]
```

```
[URL] 이거 기반으로 새 썸네일 3개 만들어줘
- 더 강렬한 느낌으로
- 문구는 짧게
```

```
2번이 마음에 드는데, 문구만 "복수의 시작"으로 바꿔줘
```

## 저장 위치

생성된 이미지는 자동으로 저장되며, 다운로드 요청 시 경로를 알려드립니다.
